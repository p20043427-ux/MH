관련근거 인증기준 2.2.1 검토책임자 병원장
[ 목 적 ]
외래환자의 요구에 맞는 서비스 제공, 적합한 진료과목 선정, 적절한 치료계획을 수립하
기 위함이다.
[ 적용범위 ]
1. 적용장소 : 외래
2. 시행자 : 의사
3. 적용범위 : 진료과 초진 환자
[ 정 의 ]
1. 외래환자 평가 : 해당과를 처음 방문하는 환자로, 본원에서 정한 초진환자를 평가하는
것을 말한다.
2, 신환(신규)환자 : 병원에 최초 내원하는 환자
3. 초진 환자 : 해당 진료과에 최초 진료 받는 환자
4. 외래환자 초기평가에 참여하는 의료진 : 외래에서 의학적인 평가를 담당하는 의사와
간호평가를 담당하는 외래 간호직원으로 정의한다.
5. 특수 환자 : 환자의요구와 특성에 따라 개별화된 진료 서비스가 필요한 환자를 말하며
본원은 신생아, 소아, 임산부를 특수 환자로 정한다.
[ 정 책 ]
1. 외래환자 초기평가 규정이 있다.
2. 타 의료기관에서 의뢰된 경우, 진료관련 정보를 확인한다.
3. 의사는 외래환자 초기평가를 수행하고 기록한다.
4. 의사는 외래 특수 환자의 초기평가를 수행하고 기록한다.
[ 절 차 ]
1. 외래환자 초기평가
가. 초기평가 대상자 : 진료과 초진 환자 ( 신환 환자, 진료과 최초 진료환자)
단, 다음과 같은 경우는 초기평가 대상에 적용되지 않는다.
1) 진료 목적이 아닌 서류상의 문제로 처음 방문하는 초진환자
2) 응급실에서 진료 후에 해당 외래를 첫 방문 시

3. 초기평가 내용
가. 공통
1) 환자가 외래로 내원한 주 호소
2) 현 병력
3) 과거력
4) 가족력
5) 신체검진
6) 추정진단
7) 치료계획
8) 통증 유무 및 양상
나. 소아청소년과, 산부인과 : 공통항목을 기본으로 하여 특수 환자의 요구와 특성에
맞도록 개별화된 초기평가를 수행하고 기록한다.
4. 초기평가 작성시한 : 환자진료 당시에 이루어지며 부득이한 사유로 보완이 필요한 경
우에는 의무기록규정에 따른다.
5. 타 의료기관에서 의뢰된 환자의 정보를 확인하는 절차
의뢰된 환자의 진료 정보(검사결과지, 영상기록물, 의사소견서 등)를 확인하고, 진료
시 정확한 환자평가를 위하여 이를 활용한다.
가. 원무팀 진료협력팀 직원은 의료기관에서 의뢰된 환자가 검사결과지, 의사소견서
등을 지참했는지 확인하고, 진료전에 외래 진료과에 제출하도록 안내한다.
나. 외래 간호팀 직원은 타 의료기관의 영상기록물(CD)을 지참했는지 확인하고, 진료
전 영상 CD 등록기에 저장하도록 환자(보호자)에게 안내한다.
다. 진료가 종료된 후 외래 간호팀 직원은 타 의료기관에서 가져온 의사 소견서, 의무
기록 등을 스캔하여 저장한다.
[ 특수환자 초기평가 ]
특수 환자는 외래 환자 초기평가를 기본으로 하되, 각 환자에 맞춤화된 평가항목으로
초기평가를 수행하고 기록한다.
1. 신생아 : 주 호소, 현 병력( Maternal Hx, Neonatal Hx), 신체검사, 추정진단, 치료계
획, 통증 유무 등을 선택하여 수행하고 기록한다.
2. 소아 : 주 호소, 현 병력, 과거력, 가족력, 소아계통문진, 신체검사, 추정진단, 치료계
획,통증유무 등을 선택하여 수행하고 기록한다.
