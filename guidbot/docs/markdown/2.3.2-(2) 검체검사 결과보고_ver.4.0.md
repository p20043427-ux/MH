[ 목 적 ]
본 규정은 의료서비스의 연속성 유지를 위해 적시에 정확한 검체검사 결과를 의료진에게
보고하며, 검사결과 이상치 발생 시 신속하게 대응하기 위함이다.
[적용범위]
1. 장 소 : 병리과
2. 대 상 : 임상병리사, 의사
[ 정 의 ]
1. 검사결과보고시간 (Turn around time, TAT) : 검사실에 검체 접수된 시간부터
결과보고 시간까지의 소요시간을 말한다.
2. 이상결과보고 (Critical value report, CVR) : 검사결과가 중대한 또는 예상하지 못한
병리결과 등 환자에 빠른 임상적 결정이 필요한 경우의 진단일 때 신속하고 정확하게
보고하는 것을 말한다.
3. 중간결과보고 : 최종진단이 지연될 때 신속한 환자치료 목적을 위해 중간결과 필요 시
임시로 결과를 보고하는 것을 말한다.
4. 결과수정 : 진단을 수정하거나 누락된 진단을 추가 또는 필요 없는 진단을 삭제하는
경우를 말한다.
[ 정 책 ]
1. 안전한 검체검사 결과보고 절차가 있다.
2. 정확하고 신속하게 검체검사 결과를 보고한다.
3. 검체검사의 이상결과(CVR)를 보고한다.
4. 검체검사의 중간결과 보고한다.
5. 보고된 검체검사 결과 변경 시 관련 의료진에게 즉시 보고한다.
[ 절 차 ]
1. 검체검사 결과보고
가. 검사 종류별 보고방법
1) 검사결과는 본원의 자인컴((OCS)프로그램을 이용하여 전산 입력 보고한다.

2) 모든 병리검사는 병리 전문의가 최종 판독, 보고함을 원칙으로 한다.
나. 검사결과 보고시간(Turn-Around Time, TAT)
1) 검사실 접수 일시부터 결과보고 일시까지를 기준으로 한다.
2) 검체검사의 TAT 지표관리는 선택한 일정 기간 내의 목표 충족 검사건수/ 전체 검사
건수를 TAT충족 비율로 나타낸다.
3) 결과 지연 건에 대해 지연 사유를 분석, 기록한다.
4) TAT의 적정성을 월별 산출하여 관리한다.

2. 이상검사 결과보고(Cirtical Value Report, CVR) 방법, 내용, 관리방법
가. 보고 대상 및 기준
1) 기준 : 결과 확인이 지연 또는 누락되어 적절한 조치를 취하지 않았을 때 환자에게
심각한 위해를 끼칠 가능성이 있다고 병리과 전문의가 판단하는 경우
2) 보고 대상
(1) Fat in colonic endoscopic polypectomies.
(2) Significant disagreement between frozen section and final diagnosis.
(3) Unexpected malignancy.

3. 중간결과 보고
가. 실시 기준 및 보고대상
1) 신속한 환자치료 결정을 위해 임상적으로 담당의의 요청이 있는 경우
2) 추가검사나 내/외부 자문으로 인하여 최종진단이 지연될 경우
3) 병리과 전문의가 중간보고할 필요가 있다고 판단하는 경우
다. 보고방법 : 병리과 전문의가 담당의에게 유선 보고하고, 담당의의 요청 시 임시 진
단 보고서로 전산 보고한다.
라. 관리방법 : 중간결과보고 관리대장에 등록번호, 환자이름, 병리번호, 접수일, 통보일,
중간결과, 담당의(수신자), 보고자 등을 기록하고 보관한다.
4. 보고된 검사결과가 변경될 경우 보고절차
가. 기준 : 보고된 검체검사의 결과변경(진단 수정/추가/삭제)이 있는 경우
나. 보고방법 : 병리과 전문의가 담당의에게 유선으로 알린다.
다. 관리방법
1) 수정진단 관리대장에 등록번호, 환자이름, 병리번호, 접수일, 처음통보일, 처음
진단, 수정통보일, 수정진단, 불일치 정도, 불일치 분류, 담당의(수신자), 보고자 등을
기록하고 보관한다.
