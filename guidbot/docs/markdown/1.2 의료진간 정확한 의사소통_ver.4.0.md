관련근거 인증기준 1.2 승인책임자 병원장
[ 목 적 ]
정확한 의사소통은 안전한 진료를 위해 매우 중요하며, 그 중 구두처방, 필요시 처방, 혼
동하기 쉬운 부정확한 처방 등 정확한 의사소통 절차를 마련하고 안전하게 관리하기 위
함이다.
[ 적용범위 ]
1. 장소 : 병동, 중환자실, 응급실, 수술실, 약제부서
2. 대상 : 의사, 간호사, 약사
3. 적용범위 :
가. 구두처방 : 응급상황 또는 수술/시술 등과 같이 처방이 불가능한 제한된 상황
나. 필요시 처방 : 각 해당 진료과에서 PRN약품신청서 작성 후 약무위원회에서 검토
결정하여 승인한 처방
[ 정 의 ]
1. 구두(전화)처방[Verbal(telephone) order]이라함은 서면 또는 전자처방 전달 시스템 등
을 사용하지 않고 구두로 전달된 처방이라한다.
2. 필요시처방[pro re nata, as needed(p.r.n)] 필요시에 사용할 수 있도록 의사가 적응증
을 명시하여 어떤 경우에 수행(투여)하라고 미리 기술해 둔 처방이라한다.
[ 정 책 ]
1. 정해진 절차에 따라 구두처방을 수행한다.
2. 필요시 처방(PRN) 을 관리한다.
3. 필요시 처방(PRN)을 안전하게 수행한다.
4. 혼동하기 쉬운 부정확한 처방 시 대처방안을 알고 수행한다.
[ 절 차 ]
[ 필요시 처방(p.r.n) 절차 ]
1. 환자확인 방법
정확한 의사소통을 위해 환자이름, 등록번호 [1.1 정확한 환자확인]규정을 따른다.
2. 구두 또는 전화처방 절차
가. 구두/전화 처방은 수술/시술 및 응급상황 등과 같이 처방이 불가능한 제한된 상황

1. 환자안전 보장활동
에서만 수행한다.
나. 구두/전화 처방하는 의사는 정확한 처방내용과 환자정보를 처방받는 간호사에게
정확하게 전달한다.
다. 정확한 환자확인
정확한 의사소통을 위해 환자이름, 등록번호 [1.1 정확한 환자확인]규정을 따른다.
의사가 환자의 등록번호를 모르는 경우 환자이름, 진단명, 환자상태로 확인한다.
라. 받아 적기 (Write down)
처방을 받은 간호사는 구두/처방 내용을 듣는 동시에 정확한 환자를 확인하고
처방 내용을 [구두/전화 처방 메모지]에 기록(write down)한다.
- 구두지시 날짜, 시간, 환자이름, 등록번호, 진단명, 진료과 등
- 약물의 경우 용량과 단위(mg, g, mEq 등), 투여경로를 정확하게 기록한다.
- 구두지시를 한 처방 의사명
- 구두지시를 받은 간호사명
라. 되읽어 확인하기 (Read-back)
구두처방을 받는 자는 받아 쓴 내용을 구두처방한 의사에게 다시 읽어 준다.
- 혼동하기 쉬운 약어, 기호를 사용하지 않고 정식 명칭을 사용
- 숫자, 익숙하지 않은 영어 의약품명의 경우 철자를 하나씩 읽어 확인
- 의약품의 용량과 단위(예시: mg, g, mEq, mVol)를 정확하게 표현
마. 확인하기 (Confirm)
처방한 지시자가 정보의 정확성을 확인한다.
바. 의사의 구두처방은 24시간 이내 전산으로 처방한다.
사. 심폐소생술 상황 등의 응급상황 시
받아적기를 생략하고 구두 지시를 받는 즉시 복창(Repeat-back) 후 이를 확인
(Confirmation) 받고 의료행위를 수행한다.
아. 구두/전화처방 수행 후 간호사는 투약 처방 , 검사/영상 처방, 기타처방 내용등과
처방 의사명을 간호기록에 기록으로 남긴다.
(예: ~처방을 ○○○의사의 구두/전화처방으로 수행함)
자. 구두/전화 처방 메모지 폐기절차
구두/전화처방 메모지는 구두지시 의사의 전산 입력 확인 후 파쇄기에 의해 폐기
한다.
3. 구두/전화 처방 시 다음의 정보는 반드시 포함한다.
가. 환자이름, 등록번호, 생년월일(주민등록상), 진단명, 처방 의사명
나. 구두/전화 처방 일시, 시각
다. 처방내용
라. 처방 의사명, 처방 받은 간호사명
마. 구두/전화 처방 시 다음의 경우는 처방할 수 없다
마약, 혈액제제, 고농도전해질(KCl, NaCl) , 헤파린, 인슐린, 백신, 항생제, 항암제
4. 진단검사의학과에서는 [구두/전화처방 메모지]로 검사 접수하면 처방내용을 정확하게
확인하고 처방 확인 후 구두/전화 메모지는 파쇄기로 폐기한다.

[ 필요시 처방(p.r.n) 절차 ]
1. 필요시 처방(p.r.n) 목록 관리
가. 필요시 처방(p.r.n)이 가능한 처방의 종류를 선정한다.
나. 처방의 종류별 가능한 목록 관리
처방의 종류 : 검사, 처치, 의약품 등
처방 불가 의약품 종류 : 전해질제제, 헤파린 주사제, 항암제류, 백신, 항생제 주사제,
경구용 항응고제, 면역억제제, 혈액제제, 마약, 향정신의약품
다. PRN목록 관리는 약무위원회에서 관리한다.
1) PRN 처방가능 목록은 약무위원회에서 PRN 처방 현황과 임상현장의 요구를 반영
하여 년 1회 검토하고 개정한다.
2) 약무위원회는 각 분야를 대표할 수 있는 의사, 간호사, 약사로 구성되어 있다.
2. 필요시 처방(p.r.n) 원칙
가. 처방수행을 위한 명확한 실시기준(사유)을 반드시 remark에 적는다.
나. 처방에 포함되어야 할 내용
1) 검사, 처치 : 수행이 필요한 환자의 상태(범위) 및 수행 내용, 횟수
2) 의약품 : 환자 상태(범위,상세 적응증1)) , 1회용량, 투여경로, 최소 투여간격,
1일 최대 투여횟수
다. 증상의 반복으로 인하여 처방이 필요한 경우 담당의에게 연락한다.
라. PRN 처방에 의한 간호수행 시 간호사는 PRN 목록에 의한 처방인지, 실시 기준
이 명확한지 확인 후에 수행하고 수행 후 PRN 처방에 의한 수행임을 간호기록에
남긴다.
마. PRN 처방은 자동 중단 처방이 적용되어 처방일이 지나면 자동으로 투약이 불가
하며 처방의 효력은 익일 자동으로 중단되어 매일 다시 처방해야 한다.
[ 부정확한 처방 절차 ]
1. 혼동하기 쉬운 부정확한 처방 유형 및 대처방안
가. 혼동하기 쉬운 부정확 처방 유형
1) 수기 처방의 경우 알아 볼 수 없는 글씨체
2) 전자 처방의 경우 처방의 의미가 명확하지 않은 상황
3) 유사발음, 용량주의, 유사코드, 유사외관
[부록 3] <혼동하기 쉬운 부정확한 약품 리스트>
1) 상세 적응증 : 측정도구 적용 또는 수치 입력이 가능한 경우

나. 혼돈하기 쉬운 부정확한 처방 방지
1) 의사, 간호사는 유사발음, 용량주의, 유사코드, 유사외관 약물의 경우 OCS 조회
목록에서 확인할 수 있다.
2) 의사는 의약품 처방 시 혼동하기 쉬운 부정확한 처방유형을 표시한 약품명을 확
인한다.
3) 간호사는 유사외관 약물의 경우 조제실 약장에 부착하여 확인한다.
4) 불가피한 수기 처방 시 약품명, 검사 및 영상 처방명, 기타 처치 처방명 등은 Full
term을 사용하여 정확하게 알아볼 수 있도록 처방한다.
5) 약제팀에서는 출력된 처방전의 약품명에 표시된 부정확한 처방유형을 확인하여
조제한다.
다. 혼동하기 쉬운 부정확한 처방 시 대처방안
1) 간호사는 처방이 부정확하거나 처방의 내용이 의문사항이 있을 경우 처방의사에
게 직접 혹은 유선으로 확인한다.
2) 처방의사는 부정확한 처방에 대하여 재검토하여 처방을 변경한다.
3) 약사는 조제절차에 따라 조제 전 처방감사 시 재확인한다.
4) 투약 단계에서 처방된 약물이 부정확하거나 불명확하다고 판단되는 경우 수행을
보류하고 처방한 의사에게 확인이 완료된 이후에 투약한다.
2. 혼동하기 쉬운 부정확한 처방확인 시 처방의사와 연락이 불가능한 경우
가. 온콜 당직의 → 해당과의 장 → 진료부장 → 병원장
나. 야간인 경우 야간당직의사에게 확인한다.
